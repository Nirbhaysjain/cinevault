# CineVault 🎬

A film tracking web app where users can discover films, log what they've watched, track their mood, and connect with other film lovers.

## Live Demo
Coming soon

## Tech Stack

**Frontend**
- React.js
- CSS
- React Router DOM

**Backend**
- Node.js
- Express.js

**Database**
- MySQL

## Features
- Browse and discover films
- Log films with mood tracking
- Add films to watchlist
- Follow other members
- Login and signup authentication

## Pages
- Landing Page
- Login / Sign Up
- Discover
- Members
- Profile (Logs + Watchlist)

## Getting Started

```bash
cd cinevault-react
npm install
npm start
```
